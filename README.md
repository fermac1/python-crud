# Python CRUD (Learning Project)

This repository is a **learning-focused Python CRUD project** built to understand core backend concepts such as **Create, Read, Update, and Delete (CRUD)** operations using Python.

The project uses:
- **Pure Python**
- **JSON file storage** (as a lightweight database)
- **CLI (Command Line Interface)** for interaction

It is intentionally simple and framework-free to focus on fundamentals.
